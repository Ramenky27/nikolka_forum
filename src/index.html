Архив бумажного форума Николки

Бумажный форум представлял собой тетрадку лежавшую в нише зала Трех Братьев в которой каждый желающий мог поделиться своими впечатлениями, контактами, оставить для других какое-то сообщение или ништяки. Форум просуществовал в системе предположительно с 2005г по 2010г. Удалось найти сканы только до 2008г. Тем не менее этот период охватывает основной пик активности Киевского дигг сообщества.
